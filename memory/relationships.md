# Relationships Memory

> L2 — Key contacts, communication styles, and relationship context.
> Cap: 6,000 chars.
>
> Populated gradually. Each person who JARVIS regularly drafts communications to or researches on your behalf gets a short block.

Entry format:

```
## <Name>
- **Relationship**: [How you know them — co-founder, client, mentor, friend, family]
- **Communication style**: [Formal / casual / terse / warm — adjust drafts accordingly]
- **Preferred channels**: [Email, Slack, text, etc.]
- **Current context**: [What you're working on together, if anything]
- **Notes**: [Anything else useful — time zone, preferences, ongoing threads]
```

Empty on first run.
