# Learnings Memory

> L2 — Self-improvement loop. Patterns, rules, and habits JARVIS has extracted from how you work. Cap: 6,000 chars.
>
> Entries accumulate when a pattern repeats 3+ times, when a session ends with an unexpected result, or when you explicitly say "save this" after a correction or validated approach.

Entry format:

```
## YYYY-MM-DD — <rule or pattern>
**Rule**: [The extracted learning in one sentence]
**Context**: [When this applies / when it doesn't]
**Evidence**: [Where this came from — which sessions, how many occurrences]
**How to apply**: [What JARVIS should do differently going forward]
```

Empty on first run — grows organically as JARVIS learns your working style.
